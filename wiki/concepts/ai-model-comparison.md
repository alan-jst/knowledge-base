## Definition
AI模型比较（AI Model Comparison）

## Key Points

- 多模型试用：建议亲自注册并试用 GPT、Claude、Gemini、Kimi、Qwen 等主流大模型
- 工程意识培养：对比同一任务（如解释代码、写单元测试）在不同模型的表现，记录哪家更稳定、谁更爱产生幻觉
- 输出质量评估：关注模型的代码生成能力、结构化输出、长文阅读、多模态支持等特性差异
- 合规使用：优先使用各平台官网订阅、教育优惠、官方 API 按量计费，避免使用二手平台共享账号（违反协议、封号风险）
- 保密要求：课题数据、未公开论文、企业敏感信息默认不进公网模型，确需使用时须按课题组规定脱敏或经导师同意
- 资源优化：利用 OpenRouter 等聚合 API 平台进行多模型对比，减少单独注册成本

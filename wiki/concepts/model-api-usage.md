## Definition
模型API使用（Model API Usage）

## Key Points

- API 密钥管理：注册并妥善保管 API Key，切勿提交到 Git 仓库，防止泄露
- 聚合平台利用：OpenRouter 等提供统一 API 入口和免费额度，适合练手「换模型、比输出」
- 条款合规：调用前阅读各模型提供方的使用条款与数据策略，确保合规使用
- 成本控制：优先使用免费额度、教育优惠、按量计费，避免不必要的订阅支出
- 安全考虑：课题数据、未公开论文、企业敏感信息不进公网模型 API，确需使用时须脱敏或经批准
- 工程实践：通过 API 调用实现自动化任务，如批量处理、对比实验、系统集成

## Description

GPT（Generative Pre-trained Transformer）是 OpenAI 开发的大语言模型系列，包括 ChatGPT 等产品，以长对话、工具调用和代码生成能力著称。

## Key Contributions

- 作为主流大模型之一，提供强大的自然语言理解和生成能力
- 核心特性：长上下文支持、工具调用（函数调用）、代码生成、多轮对话
- 工程应用：适合技术问答、代码编写、文档生成、单元测试编写等任务
- 合规使用：通过 OpenAI 官网或合规渠道访问，优先使用官方 API 按量计费
- 对比学习：在 AI 模型比较中作为基准，评估其他模型的性能和幻觉倾向

## Description

Qwen（通义千问）是阿里云开发的大语言模型系列，以开源生态和优秀的中文支持著称，提供多种尺寸的模型版本。

## Key Contributions

- 作为开源大模型代表，提供完整的开源生态和商业化 API 支持
- 核心特性：优秀的中文语言理解、开源模型权重可用、多种尺寸选择（从 0.5B 到 72B）、工具调用支持
- 工程应用：适合中文 NLP 任务、开源项目集成、定制化模型开发、成本敏感场景
- 合规使用：通过通义千问官方渠道或开源版本访问，注意开源协议和商业使用条款
- 对比学习：在 AI 模型比较中评估开源模型性能和中文任务表现

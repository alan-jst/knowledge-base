## Description

Project Gutenberg 是一个数字图书馆，提供超过 7 万本公版书（版权已过期）的免费电子版，专注于经典文学作品和学术著作。

## Key Contributions

- 作为公版书资源库，提供合法、免费的经典电子书下载
- 资源特点：专注于版权已过期的公版书，包括文学经典、历史著作、学术文献等
- 合规优势：所有资源完全合法，适合学术引用和自由使用，无版权风险
- 格式支持：提供多种电子书格式（EPUB、Kindle、纯文本等），兼容各类阅读设备
- 学术价值：作为开放获取资源的典范，支持教育研究和知识传播

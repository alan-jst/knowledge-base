## Description

OpenRouter 是一个大模型 API 聚合平台，提供多家模型（GPT、Claude、Gemini 等）的统一 API 入口和免费额度，适合模型对比和低成本试用。

## Key Contributions

- 作为 API 聚合平台，提供多家模型的统一 API 接口，简化多模型调用流程
- 成本优势：页面上标注部分模型的免费额度或低价试用，适合练手「换模型、比输出」
- 工程便利：无需为每家模型单独注册账号，一个 API Key 即可访问多种模型
- 使用规范：仍需注册并妥善保管 API Key，勿提交到 Git；调用前阅读各模型提供方的使用条款与数据策略
- 学习价值：适合初学者进行模型对比实验，了解不同模型的特性差异

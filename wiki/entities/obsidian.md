## Description

Obsidian 是一款本地优先的知识管理工具，支持双向链接、图谱视图和 Markdown 编辑，用于长期知识沉淀和个人知识库建设。

## Key Contributions

- 作为本地知识库工具，支持双向链接、图谱视图、标签系统等核心功能
- 与文件夹同步方案配合使用，实现跨设备知识管理
- 建立个人知识库：重要资料不要只放在浏览器收藏夹，在 Obsidian 中保存一份便于搜索和关联
- 学习记录管理：记录读书笔记、技术学习心得、项目总结，形成持续积累
- 工程习惯培养：材料用 Markdown 保存，便于 Git 版本管理和长期维护

## Description

X（原名 Twitter）是一个社交媒体平台，以实时短消息和工程动态分享著称，适合跟踪官方发布和技术趋势。

## Key Contributions

- 作为工程动态信息源，适合跟踪发版说明、安全公告、会议幻灯链接、技术趋势
- 信息分类：建议用列表（Lists）把关注账号分为平台官方、AI/ML 工程、语言运行时、工程实践等类别，避免时间线混乱
- 官方账号价值：关注 @GitHub、@vercel、@cursor_ai、@Docker、@Cloudflare、@flydotio、@AnthropicAI、@OpenAI、@huggingface、@GoogleDeepMind、@PyTorch、@rustlang、@golang、@nodejs、@deno_land 等获取权威信息
- 使用限制：推文不等于文档，重要参数以官网与 API 文档为准；信息极碎、节奏快，不适合替代系统学习
- 工程价值：获取第一手的产品更新、安全通告、生态动态，建立行业连接

## Description

Python 是一种高级编程语言，在 AI 工程中作为基础开发语言，用于机器学习、数据分析和脚本编写。

## Key Contributions

- 作为 AI 工程核心语言，支持机器学习框架（PyTorch、TensorFlow）、数据处理和脚本开发
- 环境管理：使用 venv 虚拟环境隔离项目依赖，会 pip install 管理包
- 学习路径：入学第一学期掌握变量、控制流、函数、文件读写、简单 json/csv 处理等基础语法
- 工程习惯：建立 requirements.txt、简单 README.md、脚本可从命令行运行的项目结构
- 与 AI 结合：用 Cursor 写 Python 脚本时，自己运行并看懂输出，把「模型写的代码」当作初稿并理解每一行
